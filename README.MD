# Golang Point Of Sale

Example Golang API backend rest implementation mini project Point Of Sale using Gin Framework and Gorm ORM Database.

## Command

- ### App Lifecyle

```sh
$ go run main.go || make go run
```

- ### Docker Lifecycle 

```sh
$ docker-compose up -d --build
```
